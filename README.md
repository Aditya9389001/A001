# How to setup
## Clone the repo

Fork and clone the repo

```bash
git clone git@github.com:Aditya9389001/A001.git
cd A001
```

## Install dependencies

```bash
npm install
```

# Finally run the Server 

```bash
nodemon server.js
```

The server will be running on `localhost:5000`
